Code and data for paper "A general theory for temperature-dependence in biology"

1) finalcode #contains the R code to run the analysis and generate: fig 1-2, fig S2-6, and tables S2-6

2) data1 #contains the data for biological variables

3) data2 #contains the data for temperature of each variable

4) reference_list.txt #contains the list of references of the papers from which was obtained the data of temperature response curves

5) database overview.txt # contains the details for each temperature response curve
